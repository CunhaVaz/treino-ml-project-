# 📊 Treino ML Project — Análise de Dados e Visualização

Este projeto foi desenvolvido no contexto de **aprendizagem de Machine Learning e Data Science**, com o objetivo de praticar:  
- **Exploração de dados** (pandas, Sweetviz)  
- **Visualização interativa** (Dash + Plotly)  
- **Preparação de dados para modelação**  
- **Integração com Git e GitHub**  

---

## 🚀 Funcionalidades

- **Relatório exploratório automático** com [Sweetviz](https://pypi.org/project/sweetviz/)  
- **Dashboard interativo em Dash** com:
  - Filtros por Canal, Produto e Ano  
  - KPIs (Total de Vendas, Margem Média, Nº Clientes)  
  - Gráficos:
    - Top 10 Clientes por Vendas  
    - Vendas ao longo do tempo (Ano-Mês)  
    - Distribuição da Margem (%)  
    - Vendas por Ano  
    - Vendas por Mês (global)  
    - Vendas por Mês no Ano selecionado  

---

## 🛠️ Tecnologias utilizadas

- Python 3.11  
- pandas, numpy  
- Plotly e Dash  
- Sweetviz  
- scikit-learn (para treino de modelos preditivos)  
- Git + GitHub  

---

## 📂 Estrutura do projeto
├── app_dash.py              # Dashboard interativo
├── analise_sweetviz.py      # Relatório exploratório automático
├── explorar_dataset.py      # Scripts de diagnóstico inicial
├── data/                    # (ignorada) Base de dados local
├── outputs/                 # Relatórios e CSVs gerados
├── reports/                 # Relatórios HTML (Sweetviz)
├── requirements.txt         # Dependências
└── README.md                # Documentação do projeto
